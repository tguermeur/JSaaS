# Firebase Functions

Ce dossier contient les fonctions Firebase pour notre application.

## Installation

```bash
npm install
```

## Développement

Pour démarrer l'émulateur Firebase Functions en mode développement :

```bash
npm run serve
```

## Déploiement

Pour déployer les fonctions sur Firebase :

```bash
npm run deploy
```

## Structure du projet

- `src/` : Contient le code source TypeScript
- `lib/` : Contient le code JavaScript compilé (généré automatiquement)
- `package.json` : Dépendances et scripts
- `tsconfig.json` : Configuration TypeScript
